# Tailwind Theme 토큰으로 Brown 팔레트 추가

## 배경

프로젝트에 따뜻한 느낌의 brown 계열 색상이 필요했지만, Tailwind CSS의 기본 팔레트에는 포함되어 있지 않았습니다.

## 구현

### 1. Theme 토큰 정의

`tailwind.config.js`에 `@theme` 지시자를 사용하여 brown 팔레트를 추가했습니다:

```js
theme: {
  extend: {
    colors: {
      brown: {
        50: '#fdf8f6',
        100: '#f2e8e5',
        200: '#eaddd7',
        300: '#e0cec7',
        400: '#d2bab0',
        500: '#bfa094',
        600: '#a18072',
        700: '#977669',
        800: '#846358',
        900: '#43302b',
      }
    }
  }
}
```

### 2. UI 컴포넌트에 적용

네비게이션과 주요 UI 컴포넌트에 brown 팔레트를 자연스럽게 적용했습니다.

## 결과

일관된 디자인 시스템을 유지하면서 프로젝트만의 따뜻한 톤을 표현할 수 있게 되었습니다.
